## Usage

1. Clone this repository
2. `composer install`
3. `cp .env.example .env`
4. `install nodejs(14.8)`
5. `npm install`
7. `copy all migrations into the main migration folder`
8. `php artisan migrate`
9. `php artisan key:generate`
